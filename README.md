# 🚀 Dijkstra's Algorithm Visualizer

An interactive web-based visualization tool that demonstrates **Dijkstra's Single Source Shortest Path Algorithm** through step-by-step animations, graph visualization, and real-time explanations.

The application helps students understand how Dijkstra's algorithm finds the shortest path from a source vertex to all other vertices in a weighted graph.

---

# 📖 Overview

Dijkstra's Algorithm is one of the most widely used graph algorithms for finding the shortest path from a source vertex to every other vertex in a graph with **non-negative edge weights**.

This project visualizes every step of the algorithm, making it easier to understand the concepts of graph traversal, priority queues, distance updates, and shortest path construction.

# 🧮 Algorithm

## Dijkstra's Algorithm

Dijkstra's Algorithm finds the shortest path from a single source vertex to every other vertex in a weighted graph with non-negative edge weights.

### Steps

1. Initialize all distances as infinity.
2. Set the source distance to zero.
3. Insert the source into the priority queue.
4. Select the vertex with the minimum distance.
5. Relax all adjacent edges.
6. Update distances if a shorter path is found.
7. Repeat until all vertices are visited.
